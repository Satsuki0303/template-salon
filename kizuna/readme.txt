== 概要 / Introduction ==

これは、個人または小規模企業向けに設計されたレスポンシブなワンページWordPressテーマです。テーマはシンプルなデザインで、カスタムカラーとフォントをサポートし、基本的なSEO最適化機能を備えています。

This is a responsive one-page WordPress theme designed for personal or small business websites. The theme has a clean design, supports custom colors and fonts, and includes basic SEO optimization features.

== インストール / Installation ==

1. テーマファイルを `/wp-content/themes/` ディレクトリにアップロードします。
2. WordPress管理画面で、"外観" -> "テーマ" に移動し、このテーマを有効化します。
3. "カスタマイズ" メニューで、必要に応じてテーマ設定を行います。

1. Upload the theme files to the `/wp-content/themes/` directory.
2. In the WordPress dashboard, go to "Appearance" -> "Themes" and activate the theme.
3. Use the "Customize" menu to adjust the theme settings as needed.
